Autentificare

Conectați-vă cu un GitHub Personal Access Token care are acces de scriere pe acest depozit. Tokenul rămâne doar în browserul dumneavoastră.

Cum obțin un token?
1. Mergeți la github.com/settings/tokens (fine-grained)
2. Generați un token nou, alegeți doar acest depozit
3. La permisiuni, dați Read & write pentru Contents
4. Copiați tokenul și lipiți-l mai sus.

Pentru un token clasic (ghp_…) este nevoie de scope-ul repo.
