# La Nico la Obor

Site de prezentare al afacerii de familie **La Nico la Obor** — legume proaspete și covrigi tradiționali de Buzău, din Piața Obor, București.

## Pagini
- `index.html` — pagina principală (despre noi, produse, hartă, recenzii, contact)
- `retete.html` — rețete tradiționale (salată de vinete, zacuscă)
- `galerie.html` — galerie foto
- `admin.html` — panou administrativ (necesită GitHub Personal Access Token)
